# Package index

## Discover and open packages

Find registered packages and select a reproducible revision.

- [`glc_explore()`](https://tscnlab.github.io/glc-dp-r/reference/glc_explore.md)
  : Explore Global Light Commons data packages
- [`glc_packages()`](https://tscnlab.github.io/glc-dp-r/reference/glc_packages.md)
  : List registered Global Light Commons data packages
- [`glc_search_packages()`](https://tscnlab.github.io/glc-dp-r/reference/glc_search_packages.md)
  : Search registered data packages
- [`glc_open()`](https://tscnlab.github.io/glc-dp-r/reference/glc_open.md)
  : Open a Global Light Commons data package
- [`glc_schema_versions()`](https://tscnlab.github.io/glc-dp-r/reference/glc_schema_versions.md)
  : Report supported GLC schema versions

## Inspect package contents

Explore datasets, files, variables, resources, and metadata.

- [`glc_summary()`](https://tscnlab.github.io/glc-dp-r/reference/glc_summary.md)
  : Summarize a Global Light Commons data package
- [`glc_resources()`](https://tscnlab.github.io/glc-dp-r/reference/glc_resources.md)
  : Inventory data-package resources
- [`glc_datasets()`](https://tscnlab.github.io/glc-dp-r/reference/glc_datasets.md)
  : Inventory datasets
- [`glc_files()`](https://tscnlab.github.io/glc-dp-r/reference/glc_files.md)
  : Inventory declared data files
- [`glc_variables()`](https://tscnlab.github.io/glc-dp-r/reference/glc_variables.md)
  : Inventory and search declared variables
- [`glc_metadata()`](https://tscnlab.github.io/glc-dp-r/reference/glc_metadata.md)
  : Load metadata resources
- [`glc_search_metadata()`](https://tscnlab.github.io/glc-dp-r/reference/glc_search_metadata.md)
  : Search metadata values or field paths

## Read and collect data

Import metadata-described files and combine compatible groups.

- [`glc_read()`](https://tscnlab.github.io/glc-dp-r/reference/glc_read.md)
  : Read metadata-described dataset files
- [`glc_collect()`](https://tscnlab.github.io/glc-dp-r/reference/glc_collect.md)
  : Collect compatible file groups
- [`extract_metadata()`](https://tscnlab.github.io/glc-dp-r/reference/extract_metadata.md)
  : Extract metadata for imported data
- [`add_metadata()`](https://tscnlab.github.io/glc-dp-r/reference/add_metadata.md)
  : Add metadata to imported data

## Download data packages

Create portable local subsets with reproducibility manifests.

- [`glc_download()`](https://tscnlab.github.io/glc-dp-r/reference/glc_download.md)
  : Download package metadata or data
